# Overview

The SORA component is a component that allows you to define a SORA flight path and calculate the iGRC for the flight path of a drone.

The component supports the following features:

- Define a flight path using a drawing tool
- Search for a location using a search tool
- Change the basemap
- Toggle the visibility of the layers
- Zoom in and out
- Locate the user's position on the map
